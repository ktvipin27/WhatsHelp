November 3, 2021

### New Features

* Support for WhatsApp Business.
* Share chat link.
* Quick message feature.
* Share App.
* Option to rate app.
* Feedback page.
* About page.

### Improvements

* Delete option in history.
* Clear history option.
* Bug fixes.
* Dark mode improved.
